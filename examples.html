Crosspuzzle

crosspuzzle is a Javascript library that can create interactive crossword/crossnumber website widgets.

How these puzzle widgets are generated

All of the above puzzle widgets were created using the crosspuzzle Javascript library, written by Matthew Scroggs. In the <head> part of the html file, we include crosspuzzle.js:

<script type='text/javascript' src='crosspuzzle.js'></script>

Each puzzle is then generated via a call to the function crosspuzzle.

An example crossword

To create the first puzzle, we first include an empty div:

<div id='crossword1'></div>

The puzzle is then created:

<script type='text/javascript'>
crosspuzzle("crossword1", {
    "grid": ["*SUM*", "SENOR", "IV*DG", "NEPAL", "*NIL*"],
    "clues": {"across": ["Add up.", "Spanish mister.", "Roman four.",
                         "Non-continuous finite element.", "Country with non-rectangular flag.",
                         "Footballer's zero."],
              "down": ["English sept.", "French one.", "Most common.", "opp ÷ hyp.",
                       "Rugeley Trent Valley.", "Greek p."]}
})
</script>

The first input to the function is the id of the empty div we created above: crosspuzzle will place the widget inside this div. The second input is Javascript Object that includes the details of the puzzle

  • the key "grid" is used to input the crossword grid as a list of rows. * is used to denote black squares
  • the key "clues" is used to input the clues, with the across and down clues given in numerical order.

An example crossnumber

To create the second puzzle, we first include an empty div:

<div id='crossnumber1'></div>

The puzzle is then created:

<script type='text/javascript'>
crosspuzzle("crossnumber1", {
    "grid": ["???*???", "?*???*?", "???*???"],
    "clues": {"across": ["2D÷2.", "1D×2.", "2D×2.", "3A×2.","3A÷3."],
              "down": ["3A÷2.", "1A×2.", "3D×1.", "2D÷1."]},
    "valid_chars": "0123456789"
})
</script>

In this puzzle, ? is used for all the white squares in this grid: this leads to a puzzle whose answer cannot be checked or revealed. The `"valid_chars"` is used to set which characters are valid for each white cell in the puzzle.
