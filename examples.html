Crosspuzzle

crosspuzzle is a Javascript library that can create interactive crossword/crossnumber website widgets.

How these puzzle widgets are generated

All of the above puzzle widgets were created using the crosspuzzle Javascript library, written by Matthew Scroggs. In the <head> part of the html file, we include crosspuzzle.js:

<script type='text/javascript' src='crosspuzzle.js'></script>

Each puzzle is then generated via a call to the function crosspuzzle.

An example crossword

To create the first puzzle, we first include an empty div:

<div id='crossword1'></div>

The puzzle is then created:

<script type='text/javascript'>
crosspuzzle("crossword1", {
    "grid": ["*SUM*", "SENOR", "IV*DG", "NEPAL", "*NIL*"],
    "clues": {"across": ["Add up.", "Spanish mister.", "Roman four.",
                         "Non-continuous finite element.", "Country with non-rectangular flag.",
                         "Footballer's zero."],
              "down": ["English sept.", "French one.", "Most common.", "opp ÷ hyp.",
                       "Rugeley Trent Valley.", "Greek p."]}
})
</script>

The first input to the function is the id of the empty div we created above: crosspuzzle will place the widget inside this div. The second input is Javascript Object that includes the details of the puzzle

  • the key "grid" is used to input the crossword grid as a list of rows. * is used to denote black squares
  • the key "clues" is used to input the clues, with the across and down clues given in numerical order.

Note that the solution of the puzzle can be viewed by viewing the source of the page (although this is more difficult than simply clicking "Reveal all entries").

An example crossnumber

To create the second puzzle, we first include an empty div:

<div id='crossnumber1'></div>

The puzzle is then created:

<script type='text/javascript'>
crosspuzzle("crossnumber1", {
    "grid": ["???*???", "?*???*?", "???*???"],
    "clues": {"across": ["2D&div;2.", "1D&times;2.", "2D&times;2.", "3A&times;2.","3A&div;3."],
              "down": ["3A&div;2.", "1A&times;2.", "3D&times;1.", "2D&div;1."]},
    "valid_chars": "0123456789"
})
</script>

In this puzzle, ? is used for all the white squares in "grid": this leads to a puzzle whose answer cannot be checked or revealed. The key "valid_chars" is used to set which characters are valid for each white cell in the puzzle.

For this puzzle, the solution cannot be found by viewing the source of the page.

Fun's word-cross

To create the second puzzle, we first include an empty div:

<div id='fun-puzzle'></div>

The puzzle is then created:

<script type='text/javascript'>
crosspuzzle("fun-puzzle", {
    "grid": ["------R------", "-----FUN-----", ...],
    "revealed": ["------?------", "-----FUN-----", "----?????----", ...],
    "clues": {"across": [":HIDDEN", "What bargain hunters enjoy.", ...],
              "down": ["To govern.", "Part of your head.", "A fist.", ...]},
    "clue_numbers": {"across": ["F–N", "2–3", "4–5", ...],
                     "down": ["1–32", "F–7", "N–8", ...]},
    "clue_number_width": "60px",
    "custom_numbering": [
        ["", "", "", "", "", "", "1", "", "", "", "", "", ""],
        ["", "", "", "", "", "", "", "", "", "", "", "", ""],
        ["", "", "", "", "2", "", "", "", "3", "", "", "", ""],
        ...
    ],
    "show_reveal_button": false })
</script>

In this puzzle:

  • - is used in the "grid" for cells that are not included
  • ":HIDDEN" is used to hide the clue for the word FUN from the list of clues
  • the key "revealed" is used to set entries that are revealed at the start of the puzzle
  • the key "clue_numbers" is used to set custom clue numbering, so that the original numbering style of provding the numbers of the first and last letters of the entry can be used
  • the key "clue_number_width" is used to set the width of the clue number column in the clue list
  • the key "custom_numbering" is used to position custom numbers in the grid
  • the key "show_reveal_button" is used to hide the reveal button (note that the solution is still visible by viewing the source of the pade)

Reference

The full list of keys that can be used in the input to crosspuzzle is given here:

KeyRequiredDefault valuePurpose
"grid"✓The puzzle grid. * can be used for black cells, - can be used for missing cells, ? can be used for other cells if check and reveal buttons are not desired
"clues"✓The puzzle clues. :HIDDEN can be used to hide a clue
"valid_chars""ABCDEFGHIJKLMNOPQRSTUVWXYZ"Valid characters for each cell
"revealed"Characters that are revealed at the start of solving
"clue_numbers"The numbers used for each clue in the clue list
"custom_numbering"The numbers that appear in the grid cells
"clue_number_width"35pxThe width of the clue column in the clue list
"clue_length_width"35pxThe width of the clue column in the clue list
"cell_size"35pxThe width and height of each cell
"show_reveal_button"trueShows/hides the reveal buttons
