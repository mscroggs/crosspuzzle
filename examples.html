Crosspuzzle

crosspuzzle is a Javascript library that can create interactive crossword/crossnumber website widgets.

How these puzzle widgets are generated

All of the above puzzle widgets were created using the crosspuzzle Javascript library, written by Matthew Scroggs. In the <head> part of the html file, we include crosspuzzle.js:

<script type='text/javascript' src='crosspuzzle.js'></script>

Each puzzle is then generated via a call to the function crosspuzzle.

An example crossword

To create the first puzzle, we first include an empty div:

<div id='crossword1'></div>

The puzzle is then created:

<script type='text/javascript'>
crosspuzzle("crossword1", {
    "grid": ["*SUM*", "SENOR", "IV*DG", "NEPAL", "*NIL*"],
    "clues": {"across": ["Add up.", "Spanish mister.", "Roman four.",
                         "Non-continuous finite element.", "Country with non-rectangular flag.",
                         "Footballer's zero."],
              "down": ["English sept.", "French one.", "Most common.", "opp ÷ hyp.",
                       "Rugeley Trent Valley.", "Greek p."]}
})
</script>

The first input to the function is the id of the empty div we created above: crosspuzzle will place the widget inside this div.
